📊 SALES & SUPPORT ANALYTICS DASHBOARD (Power BI)
📌 Project Title

End-to-End Sales and Support Analytics Dashboard using Power BI and MySQL

🧠 Project Overview

This project is a business intelligence solution built using Power BI and MySQL to analyze sales performance and customer support operations in a single unified dashboard.

It helps stakeholders monitor:

Revenue and profit trends
Product and city performance
Customer support ticket status
Issue types and resolution efficiency

The goal is to improve data-driven decision making and operational efficiency.

🛠 Tools & Technologies Used
Power BI Desktop
MySQL Database
DAX (Data Analysis Expressions)
Power Query (Data Transformation)
📊 Data Sources

Two main tables were used:

1. Sales Table

Contains:

Order details
Product information
Quantity, price, cost
City and customer data
2. Support Tickets Table

Contains:

Ticket ID
Issue type
Priority (P1, P2, P3)
Status (Open/Closed)
Resolution time
📈 Key KPIs (Measures Created using DAX)
💰 Total Revenue
Calculated using quantity × price
💵 Total Profit
Calculated using quantity × (price - cost)
🎫 Total Tickets
Count of all support tickets
🔴 Open Tickets
Filtered count of unresolved tickets
⏱ Average Resolution Time
Average time taken to resolve tickets
📊 Dashboard Features
📌 Sales Dashboard
Revenue by City
Profit by City
Top Performing Products
Sales Trend Analysis
City-based filtering (Slicers)
🎫 Support Dashboard
Ticket Status Distribution (Open vs Closed)
Priority Breakdown (P1, P2, P3)
Issue Type Analysis
Detailed Ticket Table View
SLA Monitoring
🔍 Key Insights
Identified top revenue-generating cities
Found best-selling products
Detected high-frequency customer issues
Tracked open tickets and SLA delays
Improved visibility of business performance
